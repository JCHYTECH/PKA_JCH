# Gouvernance presentation — Nuances x Arteon

## Regle de base

Tout support de presentation produit pour `Nuances` doit s'appuyer en premier sur :

- la charte graphique `Arteon`
- les layouts de slides deja definis
- les templates et mockups deja valides dans `03_deck_system/`

Autrement dit :

- on ne recommence pas un deck "a blanc" si une base visuelle existe deja ;
- on adapte le contenu au systeme visuel avant d'inventer une nouvelle forme ;
- toute exception doit etre motivee par un besoin reel, pas par commodite.

## Regle d'usage

### Si le support est analytique

Utiliser en priorite la logique `Horizon`.

### Si le support est narratif ou de posture

Utiliser `Aurel` pour l'ouverture, la vision, certaines transitions et les slides de conviction.

### Si le support est statutaire ou de cloture

Utiliser `Nocturne` avec parcimonie pour la cover, les intercalaires et la conclusion.

## Reflexe de production

Avant de produire un nouveau deck ou une presentation :

1. verifier la charte et les variantes de layout ;
2. choisir le systeme visuel adapte ;
3. ne produire qu'ensuite le contenu slide par slide.

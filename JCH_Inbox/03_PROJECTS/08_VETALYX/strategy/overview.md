# Projet - Vetalyx

## 1. Résumé exécutif

**Vetalyx** est un projet de développement et de structuration d'une activité de diagnostics vétérinaires, avec une orientation vers les tests d'allergie pour animaux de compagnie.

Cette note est le hub central du projet. Elle doit rester lisible et ne pas contenir tous les détails. Les informations spécialisées doivent être placées dans les notes pivots.

## 2. Objectif

Structurer le lancement, le positionnement, les documents, les contrats, la réglementation, le site web, les contacts et les décisions du projet Vetalyx.

# VETALYX / JCHYTECH — Index projet

Dernière mise à jour : 2026-04-29

## Description
Initiative de lancement sur le marché européen du diagnostic d'allergies aux animaux de compagnie.
Fournisseurs : Zheda Dixun (CN) et partenaires. Marchés cibles : France, Italie, Espagne, Allemagne, Belgique.

# VETALYX - Protocole de validation clinique KOL

## 1. Synthese executive

Objectif : proposer a deux opinion leaders veterinaires un protocole simple, credible et compatible avec la routine clinique pour evaluer les tests rapides d'allergie IgE VETALYX chez le chien et le chat, tout en constituant des echantillons conserves pour comparaison avec des tests de reference type ELISA, immunoblot ou microarray.

Contacts cibles issus du fichier projet :

| Contact | Organisation | Role propose |
|---|---|---|
| Dr Morgane Cannone | Maisons-Alfort | KOL France, dermatologie/allergologie, site clinique coordinateur possible |
| Dr Billen | ULg | KOL Belgique/ULg, site clinique universitaire ou referent |

Position recommandee : etude prospective observationnelle multicentrique, non interventionnelle, sur animaux de clients, avec prelevements realises lors d'une consultation dermatologique ou respiratoire justifiee cliniquement.

Decision importante sur le buvard : le dried blood spot / buvard est prometteur pour reduire les contraintes logistiques et le cout, mais ne doit pas etre le support primaire de validation tant que la correlation avec serum/plasma n'a pas ete demontree pour les IgE chien/chat. Il doit etre integre comme **sous-etude de faisabilite analytique**, avec prelevement pairé serum/plasma + buvard.

Phrase de position a utiliser avec les KOL :

> Le DBS sera evalue comme matrice alternative de conservation et de transport, en comparaison appariee avec serum/plasma, pour determiner sa concordance analytique avec les methodes IgE de reference.

## 2. Hypotheses a valider

### Hypothese clinique

Le test rapide VETALYX peut etre utilise en routine comme outil de triage/orientation chez chiens et chats presentant des signes compatibles avec allergie environnementale, dermatite atopique, hypersensibilite a la puce, ou suspicion alimentaire.

### Hypothese analytique

Les resultats du test rapide ont une concordance acceptable avec un test de reference laboratoire sur serum/plasma pour les grandes familles d'allergenes.

### Hypothese logistique

Le buvard peut permettre la conservation et l'expedition simplifiee d'echantillons pour analyses IgE de reference, mais uniquement si une etude de pontage demontre une concordance suffisante avec le serum/plasma.

## 3. Design propose

| Element | Proposition |
|---|---|
| Type d'etude | Prospective, observationnelle, multicentrique, routine clinique |
| Sites | 2 KOL : Maisons-Alfort + ULg |
| Population | Chiens et chats de clients presentes pour prurit, dermatite, otite chronique, suspicion atopie, suspicion allergie alimentaire ou signes respiratoires allergiques |
| Intervention | Test rapide IgE VETALYX realise en consultation ou juste apres prelevement |
| Comparateur | Test de reference laboratoire sur serum/plasma : ELISA, immunoblot, Dot Array/microarray, ou plateforme allergologique reconnue |
| Sous-etude DBS | Buvard dried blood spot preleve en parallele du serum/plasma |
| Duree pilote | 3 a 4 mois de recrutement + 1 mois analyses reference |
| Nature du risque | Minimal si le prelevement sanguin est deja cliniquement justifie ou limite en volume |
| Positionnement | Faisabilite clinique + concordance analytique, pas validation definitive de diagnostic allergologique |

## 4. Objectifs

### Objectif primaire

Evaluer la concordance entre le test rapide VETALYX et un test de reference IgE sur serum/plasma, par famille allergenique.

### Objectifs secondaires

- Mesurer le taux de resultats invalides ou indeterminés.
- Evaluer la facilite d'utilisation en consultation : temps de manipulation, lisibilite, acceptabilite clinique.
- Evaluer l'impact sur la decision clinique : elimination parasite, panel complet, immunotherapie, regime d'eviction, traitement symptomatique.
- Comparer les resultats serum/plasma et buvard DBS sur un sous-ensemble pairé.
- Estimer les contraintes logistiques : temps de prelevement, stockage, expedition, cout par cas.

## 5. Population et recrutement

### Inclusion

- Chien ou chat de client, age > 6 mois.
- Consultation pour au moins un signe compatible :
  - prurit chronique ou recurrent ;
  - dermatite, erytheme, alopecie auto-induite, dermatite miliaire, otite chronique ;
  - suspicion de dermatite atopique ou feline atopic skin syndrome ;
  - suspicion d'allergie alimentaire avec manifestation cutanee ou digestive ;
  - signes respiratoires compatibles chez le chat, si le clinicien juge pertinent.
- Consentement ecrit du proprietaire.
- Possibilite de prelever un volume sanguin compatible avec le poids et l'etat de l'animal.

### Exclusion

- Animal instable ou prelevement non justifie cliniquement.
- Traitement immunosuppresseur lourd recent si le KOL estime qu'il invalide l'interpretation.
- Infection cutanee non controlee si elle empeche l'interpretation clinique immediate.
- Refus du proprietaire pour conservation ou expedition d'echantillon.
- Volume sanguin insuffisant.

## 6. Stratification clinique

Chaque animal doit etre classe par le veterinaire dans une categorie principale avant connaissance du test de reference :

| Code | Groupe clinique | Exemple |
|---|---|---|
| D-ENV | Chien allergie environnementale/atopie suspectee | prurit saisonnier/non saisonnier, dermatite atopique |
| D-DERM | Chien dermatite allergique dominante | puce, otite/dermatite chronique, Malassezia associee |
| D-FOOD | Chien suspicion alimentaire | prurit non saisonnier, troubles digestifs associes |
| C-ENV | Chat allergie environnementale/FASS suspectee | prurit cervico-facial, dermatite miliaire, alopecie auto-induite |
| C-DERM | Chat dermatite allergique dominante | puce, insectes, lesions cutanees allergiques |
| C-FOOD | Chat suspicion alimentaire | prurit/non saisonnier, digestif, historique alimentaire compatible |
| C-RESP | Chat signe respiratoire compatible | asthme/allergie respiratoire suspectee |

## 7. Workflow en consultation

1. Consultation routine par le KOL ou son equipe.
2. Verification inclusion/exclusion.
3. Information proprietaire et signature du consentement.
4. Score clinique minimal :
   - chien : prurit 0-10, CADESI-4 simplifie si faisable, otite oui/non, puce oui/non ;
   - chat : prurit 0-10, pattern lesionnel, puce oui/non, respiratoire oui/non.
5. Prelevement sanguin.
6. Realisation du test rapide VETALYX.
7. Preparation serum/plasma pour test de reference.
8. Preparation buvard DBS si consentement specifique obtenu.
9. Enregistrement des resultats rapides et de la decision clinique initiale.
10. Expedition des echantillons de reference.
11. Saisie du resultat laboratoire lorsqu'il est disponible.

- Test rapide : selon IFU fabricant.
- Serum/plasma reference : viser 0,5 a 1,0 mL si possible.
- Buvard : 3 a 5 spots de sang total. Option standard pratique : 50-75 uL par spot ; option analytique controlee : 20 ou 40 uL pipetes par spot, en ligne avec des travaux veterinaires sur DBS chien/chat.

## 9. Sous-etude buvard / dried blood spot

### Statut du DBS dans le protocole

Le DBS est un **sous-protocole analytique individualise**, separe de la validation clinique principale. Il doit etre discute et valide avec le partenaire chinois avant toute utilisation comme matrice de routine.

Le protocole principal repond a la question : le test rapide VETALYX est-il utilisable en conditions cliniques de routine ?

Le sous-protocole DBS repond a une question differente : le buvard peut-il devenir une matrice fiable de conservation, transport et test IgE, apres comparaison pairée avec serum/plasma ?

Le succes ou l'echec du DBS ne doit donc pas bloquer l'evaluation clinique du test rapide. En revanche, si le DBS est valide avec le partenaire chinois, il devient un levier majeur pour scaler la biobanque et les tests de reference.

### Rationale

Le buvard peut reduire les contraintes de centrifugation, chaine froide, transport et stockage. C'est attractif pour un reseau de cliniques veterinaires, surtout si les echantillons doivent etre conserves ou envoyes plus tard pour analyses IgE.

### Limites scientifiques connues

- Les IgE sont plus fragiles que d'autres immunoglobulines et sensibles a la chaleur.
- Le DBS introduit des variables : hematocrite, volume depose, homogeneite du spot, temps de sechage, humidite, extraction, interference des cellules lysees.
- Les preuves humaines existent pour IgE totale et immunoglobulines sur DBS, mais la preuve specifique chien/chat pour allergen-specific IgE est insuffisante.
- Les resultats DBS doivent donc etre compares au serum/plasma avant usage decisionnel.

### Design de pontage DBS

| Element | Proposition |
|---|---|
| Sous-population | minimum 60 paires serum/DBS chien + 30 paires serum/DBS chat si la phase KOL le permet ; minimum absolu 30 a 40 animaux pour faisabilite |
| Especes | analyser chien et chat separement |
| Analyse | meme panel IgE sur serum/plasma et eluats DBS si laboratoire compatible |
| Conditions | depot pipete si possible, DBS seche 2-4 h a temperature ambiante controlee, sachet barriere + dessicant + indicateur humidite |
| Controle qualite | photo ou scan du spot, volume depose, heure depot, heure ensachage, temperature, humidite si possible |
| Stockage court terme | 2-8 C des ensachage ; expedition <48 h sous froid si possible ; documenter toute deviation |
| Stockage long terme | -20 C par defaut ; -80 C si biobanque disponible |
| Endpoints | taux d'extraction exploitable, concordance positif/negatif, correlation semi-quantitative, taux invalides |

### Chaine froide DBS retenue

Decision projet : conserver les buvards a basse temperature. La temperature ambiante doit etre limitee au sechage initial et aux manipulations courtes documentees.

SOP recommandee :

1. Deposer le sang sur carte DBS avec volume controle si possible.
2. Laisser secher horizontalement 2-4 h a temperature ambiante, a l'abri du soleil, poussiere, aerosols et humidite.
3. Ensacher individuellement avec dessicant et indicateur d'humidite.
4. Placer a 2-8 C le jour meme, idealement immediatement apres ensachage.
5. Pour conservation >72 h : transferer a -20 C.
6. Pour biobanque ou reanalyse future : privilegier -80 C si accessible.
7. Eviter les cycles gel/degel ; sortir uniquement les cartes necessaires a l'analyse.
8. Tracer toutes les temperatures connues : sechage, refrigeration, congelation, transport, reception.

### Tests de stabilite DBS recommandes

Comme la [[strategie]] retenue est basse temperature, les tests de stabilite doivent d'abord documenter la performance dans les conditions prevues. Sur un sous-ensemble, si le laboratoire accepte les eluats DBS :

- J0 / J3 / J7 / J14 / J28 ;
- 2-8 C versus -20 C ;
- temperature ambiante seulement comme condition de stress/deviation, pas comme condition nominale ;
- comparaison full spot versus punch si volume controle ;
- impact hematocrite si disponible dans la biologie routine ;
- analyse des faibles positifs separee, car c'est la zone la plus fragile.

Interpretation : les seuils serum/plasma ne doivent pas etre transferes automatiquement au DBS. Le DBS doit recevoir ses propres seuils ou classes d'interpretation.

### Decision go/no-go DBS

Go DBS si :

- taux d'echantillons exploitables >90 % ;
- concordance positif/negatif avec serum/plasma >=80 % par grandes familles ;
- pas de biais majeur lie au site ou au temps d'expedition ;
- protocole de sechage/stockage facilement executable par assistantes veterinaire.

No-go DBS comme support principal si :

- discordance elevee sur faibles positifs ;
- degradation observee malgre conservation basse temperature, ou degradation importante apres deviation courte a temperature ambiante ;
- extraction non reproductible ;
- impossibilite laboratoire d'analyser IgE chien/chat sur eluats DBS.

| Chats cas allergie suspectee | 20 |
| Controles cliniques non allergiques si disponibles | 20 |

Cette version permet surtout une validation d'usage, de logistique et de concordance exploratoire.

### Version recommandee KOL

| Chats cas allergie suspectee | 40 |

Repartition cible : environ 80 animaux par KOL. Cette taille ne prouve pas une performance reglementaire definitive, mais elle donne une base defendable pour decider go/no-go produit, estimer la concordance par famille allergenique, mesurer la faisabilite routine et evaluer le buvard.

|---|---:|
| Cas chiens/chats allergie suspectee | 80 a 100 |
| Controles chiens/chats | 20 |
| Total | 100 a 120 |

Cette version est acceptable pour une phase KOL pilote avant extension multicentrique.

## 11. Endpoints et analyses

### Endpoints analytiques

- Concordance globale rapide vs reference : positif/negatif par ligne.
- Positive percent agreement et negative percent agreement.
- Cohen kappa par famille si effectifs suffisants.
- Sensibilite, specificite, VPP, VPN exploratoires avec IC 95 %, uniquement si le diagnostic composite est assez robuste.
- Taux de resultats invalides.
- Concordance serum/plasma vs DBS dans la sous-etude.
- Bland-Altman si resultats DBS et serum sont quantitatifs.

### Endpoints cliniques

- Temps total ajoute a la consultation.
- Facilité d'utilisation notee par le veterinaire ou ASV : 1-5.
- Lisibilite du resultat : 1-5.
- Influence sur decision clinique : oui/non et type de decision.
- Acceptabilite proprietaire : refus, accord test seul, accord test + conservation echantillon.

### Analyses prudentes a eviter

- Ne pas presenter le test comme diagnostic definitif d'allergie alimentaire.
- Ne pas utiliser le test de reference IgE comme gold standard absolu de maladie allergique.
- Ne pas tirer de conclusions par allergene individuel si la ligne est un mix.

## 13. Points contractuels et ethico-reglementaires

- Avis comite ethique / institutionnel a demander si les sites universitaires l'exigent.
- Consentement proprietaire explicite : le consentement aux soins ne suffit pas pour une etude ou conservation d'echantillon.
- Donnees pseudonymisees : animal ID, espece, age, sexe, race, signes, traitements, resultats ; pas de donnees personnelles proprietaire dans la base centrale sauf necessite.
- RGPD : definir responsable de traitement, finalites, duree de conservation, droits des proprietaires.
- Propriete des echantillons : preciser si les echantillons residuels appartiennent au promoteur, au site, ou sont detruits apres analyse.
- Publication : prevoir droit de revue des KOL, mais conserver la capacite d'utiliser les resultats agreges.
- Indemnite : separer honoraire de coordination, compensation par cas inclus, et remboursement des consommables/transport.
- Transport : faire valider par le laboratoire/transporteur la classification exacte des echantillons ; selon contexte, les echantillons animaux peuvent relever d'un envoi "Exempt Animal Specimen" ou UN3373 Biological Substance Category B.

## 15. Proposition de phasage

### Phase 0 - Alignement KOL et laboratoire

Duree : 2 a 3 semaines.

- Valider les panels et lignes testees.
- Choisir le test de reference.
- Verifier compatibilite serum/plasma/DBS avec le laboratoire.
- Finaliser CRF, consentement et SOP.

### Phase 1 - Pilote routine

Duree : 6 a 8 semaines.

- 20 a 30 animaux.
- Objectif : faisabilite, temps consultation, taux echantillons complets, premiers accords rapide/reference.
- DBS obligatoire sur tous les cas si techniquement faisable.

### Phase 2 - Validation prospective

Duree : 3 a 4 mois.

- Extension a 80-120 animaux.
- Analyse par espece et par famille allergenique.
- Rapport final KOL.

Objet : Proposition de protocole pilote - validation clinique routine test rapide IgE chien/chat VETALYX

Bonjour Dr [Nom],

Nous preparons un protocole pilote multicentrique visant a evaluer en conditions de routine clinique un test rapide d'orientation IgE chez le chien et le chat, avec comparaison a un test de reference laboratoire et conservation d'echantillons.

L'objectif n'est pas de remplacer le diagnostic dermatologique ni les panels de reference, mais d'evaluer si un test rapide peut aider le veterinaire a orienter plus tot les cas prurigineux/allergiques et a selectionner les animaux justifiant une investigation approfondie.

Nous souhaitons vous proposer un role de KOL/site clinique pour :

- relire et ajuster le protocole ;
- inclure des cas vus en routine ;
- superviser l'interpretation clinique ;
- participer a l'analyse des resultats et a un eventuel rapport scientifique/technique.

Une sous-etude evaluera aussi la faisabilite du buvard type dried blood spot pour conserver et expedier des echantillons IgE, mais uniquement en comparaison pairée avec serum/plasma.

Seriez-vous disponible pour un echange de 30 minutes afin de valider l'interet clinique, les contraintes de terrain et les conditions pratiques d'une collaboration ?

## 3) Comment cadrer votre demande (pour recevoir des offres comparables)
Lorsque vous contactez ces spécialistes, donnez un **cahier des charges court (2–4 pages)** avec :

1) **Formes galéniques** : poudre / gélules / comprimés / liquides / stick packs, et **présence d’allergènes** (ex. lait/soja) ou d’extraits à odeur forte.  
2) **Capacité** : lots/jour, cadence, nb. opérateurs.  
3) **Niveau “salle propre” visé** : beaucoup de compléments alimentaires ne nécessitent pas ISO 7 partout ; on fait souvent un **zoning hygiénique** (zones “propres” + zones “poussières” confinées). Si vous visez une classification ISO, référez-vous à la logique ISO 14644 (classification particulaire). citeturn0search6  
4) **Flux** (indispensable) :  
   - flux matières premières → quarantaine → pesées → mélange → conditionnement → libération → expédition  
   - flux personnel (vestiaires, sas, lavage mains, EPI)  
   - flux déchets / retours / non-conformes  
5) **Contraintes bâtiment** : surface disponible, hauteur, dalle, quais, réseaux existants.  
6) **Utilités** : air comprimé, vapeur éventuelle, eau purifiée si besoin, aspiration poussières (ATEX si poudres fines), CVC (temp/hygro).  
7) **Approche qualité** : HACCP, GMP “food supplement” interne, exigences clients (ex. grands comptes) et **niveau de qualification** attendu (DQ/IQ/OQ/PQ si vous voulez pousser la rigueur).

Parfait — avec **conditionnement comprimé / gélule / blister / flacon** et possibilité **fabrication interne**, l’implantation devient plus exigeante : on passe d’un simple atelier de conditionnement à un **atelier de production complète** (pesée → granulation → compression/capsulage → enrobage si nécessaire → contrôle → conditionnement). Je vais structurer ce qu’il vous faut pour avancer **rapidement et professionnellement**.

---

## 1) Résumé des exigences techniques selon vos formats

| Étape | Exigences infrastructure | Contraintes strictes |
|-------|--------------------------|-----------------------|
| **Pesée** | Espace fermé, plancher lisse, aspiration poussières | ATEX, confinement poudres |
| **Granulation / Mélange** | HVAC dédié, flux positif/contrôlé | ISO 7/8 selon process |
| **Compression / Capsulage** | Zones vibration, bruit | Hottes de confinement |
| **Blister** | Zone propre, ventilation | Contrôle particules |
| **Conditionnement flacons** | Zone moins stricte | Flux matières |
| **Contrôle qualité interne** | Laboratoire dédié | HACCP / GMP |
| **Stock matières & produits finis** | Zoning séparé, traçabilité | FIFO, ERP ou WMS |
| **Expédition / Quarantaine** | Accès distinct | Séparation Q/PF |

---

## 2) Niveau de salles propres recommandé

Pour une **ligne complète avec production de comprimés/gélules**, on recommande :

- **ISO 8** (classe D) pour zones de production générales.  
- **ISO 7** (classe C) pour zones critiques (pesée + proximité machines de compression/blister).  
- **Zoning hygiénique strict** ailleurs (vestiaires, sas, circulation, magasin).

> Cela garantit un environnement maîtrisé pour les comprimés/gélules et réduit les risques de contamination croisée.  
> Les exigences exactes dépendront de vos matières premières (poussières fines, allergènes).

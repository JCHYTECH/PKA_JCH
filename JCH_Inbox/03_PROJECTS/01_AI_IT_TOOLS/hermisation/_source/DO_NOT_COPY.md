# Do Not Copy

The [[Hermisation]] project must not copy these patterns from Hermes-like systems:

- pseudo-AGI framing;
- autonomous decision-making for critical actions;
- opaque framework dependency;
- agent proliferation without role clarity;
- unrestricted filesystem writes;
- uncontrolled memory growth;
- always-on automation before manual validation;
- external network exposure without security review;
- storing secrets in Markdown;
- replacing [[Dobby]] as orchestrator.

## Rule

Each added layer must have a clear purpose, an owner, a rollback path, and a human-readable operating procedure.

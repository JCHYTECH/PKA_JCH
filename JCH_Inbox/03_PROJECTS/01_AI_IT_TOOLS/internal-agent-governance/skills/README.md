# Registre de skills internes

Statut: `draft`

Ce dossier contiendra les procedures reutilisables validees par JCH.

## Statuts de skill

- `draft`: proposition non active.
- `approved`: validee, prete a utiliser.
- `active`: utilisee regulierement.
- `retired`: conservee mais desactivee.

## Regle

Une skill peut etre proposee par l'assistant, mais elle ne devient pas `approved` ou `active` sans validation explicite de JCH.

| Skill | Statut | Usage |

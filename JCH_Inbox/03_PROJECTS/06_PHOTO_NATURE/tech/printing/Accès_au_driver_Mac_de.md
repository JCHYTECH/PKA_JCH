Je dois d'abord clarifier quelque chose : l'**ET8550** est une imprimante Epson, et les drivers d'imprimante sous macOS fonctionnent de manière intégrée au système.

Voici comment cela fonctionne réellement :

## 1. **Via Préférences Système (Réglages) macOS** ✅ Recommandé
C'est le **meilleur endroit** pour :
- Installer/mettre à jour le driver
- Accéder aux **paramètres avancés** de l'imprimante (qualité, type de papier, profils ICC, gestion couleur)
- Configurer les préférences par défaut
- Gérer les files d'impression

**Chemin** : `Réglages Système → Imprimantes & Scanners → Sélectionner ET8550 → Options & Fournitures`

## 2. **Via commande "Imprimer" dans Photoshop/Lightroom** 
C'est un accès **secondaire/rapide** pour :
- Imprimer directement un projet
- Accéder à certains paramètres spécifiques de ce logiciel
- Vérifier la mise en page avant impression

**Mais** : cela ne donne accès qu'aux paramètres d'impression du document actuel, pas aux configurations profondes du driver.

---

## **Mon conseil pour toi (photographe) :**
1. **D'abord** → Configure le driver via **Réglages Système** (couleur, profil, qualité maximale)
2. **Ensuite** → Utilise Photoshop/Lightroom pour imprimer avec tes paramètres de document spécifiques

C'est particulièrement important si tu travailles sur la **gestion des couleurs** pour tes tirages wildlife photo.

Ah, c'est un problème courant ! Plusieurs raisons possibles :

## **Vérifications à faire :**

1. **Le driver Epson est-il correctement installé ?**
   - Va dans : `Réglages Système → Imprimantes & Scanners`
   - Supprime l'ET8550 (clic droit → Supprimer)
   - Ajoute-la de nouveau (clic "+" → sélectionne ET8550)
   - Cela force macOS à télécharger/réinstaller le driver

2. **Le driver Epson complet n'est peut-être pas installé**
   - macOS propose souvent juste un driver générique
   - **Télécharge le driver complet** depuis le site Epson officiel :
     - [Epson Support - ET8550](https://epson.sn) (cherche ton modèle exact)
     - Installe le pack complet (pas juste l'imprimante)

3. **Une fois le driver complet installé**, clique sur l'ET8550 dans Réglages Système, puis cherche un bouton comme :
   - **"Options & Fournitures"** ou
   - **"Utilitaire"** ou
   - **"Options avancées"**

---

## **Alternative directe (souvent plus efficace) :**
Cherche dans **Applications → Utilitaires** ou **Applications → Epson** s'il y a une app dédiée style :
- **"Epson Printer Utility"** ou
- **"Epson Status Monitor"**

Ces apps donnent accès à TOUS les paramètres du driver.

## A. Add names *on* the picture (using “Insert Text”)

This will burn the text into the image (good for printing, sharing, etc.).

1. **Open the photo in digiKam**
   - Start digiKam.
   - In the album view, find your group picture.
   - Double-click it to open, then click **“Edit”** (or press **F4**) to open the **Image Editor**.

2. **Open the “Insert Text” tool**
   - In the Image Editor top menu, go to:  
     **Decorate → Insert Text**. citeturn1view0  

3. **Type the first name**
   - A dialog appears.
   - In the text box, type the person’s name (e.g. `Daniele`).
   - Choose **Font**, **Size**, and **Color**:
     - For a group photo, a **simple sans-serif** font (e.g. DejaVu Sans) works well.
     - Use **white text with a black border or semi-transparent background**, so it stays readable on any background. citeturn1view0  

4. **Place the text on the photo**
   - Click on the image where you want the name to appear (under the person, or slightly to the side of the head).
   - You can **drag** the text with the mouse to reposition it.
   - If needed, adjust rotation, background, and border in the dialog.

5. **Repeat for each person**
   - Confirm the first name with **OK** (or “Apply” depending on version).
   - Reopen **Decorate → Insert Text** for the next person, and so on.
   - It’s usually clearer if:
     - all names have **same font and size**,  
     - alignment is consistent (e.g. centered under each person).

6. **Save without destroying your original**
   - When you’re done, go to **File → Save As…**
   - Save the edited picture as a **new file** (e.g. `group-with-names.jpg`).
   - This way you keep the original photo untouched.  
   - Remember: once saved, the text is part of the image and **can’t be edited** later like normal text. citeturn1view0  

---

## B. Alternative: make a “legend” instead of names on faces

If the image is very busy, another nice option:

1. Use **Insert Text** once, to write a **legend** at the bottom, for example:  
   `Front row (left→right): Daniele, Alessandro, ...`  
   `Middle row: ...`  
   `Back row: ...`

2. Optionally, you can:
   - Add **numbers** (1, 2, 3, …) near each person’s head.
   - Then, in the legend text, map: `1 = Daniele, 2 = Alessandro, ...`

This keeps the image clean while still explaining who is who.

---

## C. If you *don’t* want text burned into the pixels

If your goal is just to **know who is who inside digiKam** (not on the exported JPEG):

- Use **People / Face Tags**:
  - In the left sidebar, go to **People**.
  - Let digiKam detect faces, or right-click → **Add a Face Tag** and draw a box on each face, then type the person’s name. citeturn0search7turn0search17  
- This stores the names as **metadata** and tags, not visible on the picture itself.
- Some versions / workflows allow using these tags when exporting, but it’s less straightforward and sometimes version-dependent. citeturn0search4turn0search18

# ✔️ Procédure détaillée dans digiKam  
*(pour ajouter un nom au-dessus de chaque tête, rangée par rangée)*

## 1. Ouvrir la photo dans l’éditeur digiKam
1. Sélectionnez la photo → double-clic.  
2. Cliquez **“Edit”** ou pressez **F4** pour ouvrir l’**Image Editor**.

---

# ✔️ Pour chaque personne : “Insert Text” au bon endroit

## 2. Ouvrir l’outil “Insert Text”
Menu : **Decorate → Insert Text**

Réglages recommandés :
- **Font** : Sans-serif (DejaVu Sans, Noto Sans, Arial-like)  
- **Taille** : 24–34 suivant la résolution  
- **Couleur** : **Blanc**  
- **Contour** (“Stroke”) : **Noir 2–3 px**  
  → garantit une bonne lisibilité sur tout arrière-plan  
- **Fond** : Aucun (ou léger fond semi-transparent si nécessaire)

---

# ✔️ 3. Placement (important avec 4 rangées)
Comme vous souhaitez **au-dessus de la tête**, voici la méthode pour garder la photo propre :

### 🔹 Rangée 1 (avant plan, tout devant)
- Cliquez juste **au-dessus des cheveux** de chaque personne.  
- Gardez les noms **légèrement plus bas** que les rangées derrière eux pour éviter le chevauchement.

### 🔹 Rangée 2
- Placez les noms **un cran au-dessus**, mais toujours assez proche des têtes.  
- Distance verticale approximative : 20–40 px selon la hauteur de la photo.

### 🔹 Rangée 3
- Montez encore le texte.  
- Attention à ne pas coller trop au bord supérieur de l’image.

### 🔹 Rangée 4 (dernière tout en haut)
- Placez les noms **très proche du haut de la tête**, mais en laissant :
  - au moins **1 à 2 cm visuel** de marge avec le bord du cadre.

🎯 **Astuce pro** : activez temporairement la grille d’alignement dans digiKam :  
Menu → **View → Show Grid** → permet d’aligner les noms horizontalement pour chaque rangée.

---

# ✔️ 4. Répéter sans casser le style
Pour chaque nouveau nom :
1. Refaites **Decorate → Insert Text**
2. Gardez *exactement* les mêmes réglages de police
3. Placez précisément sur la rangée correspondante

Cela garantit une **cohérence visuelle** (indispensable sur une photo de groupe).

---

# ✔️ 5. Sauvegarder proprement
- Menu : **File → Save As…**
- Nom conseillé : `group-photo-with-names.jpg`  
- Gardez l’original intact.

Légende propre et photo non polluée.

Pour **injecter automatiquement les noms dans digiKam**, il y a **deux méthodes possibles**, mais il faut être clair :

👉 **digiKam ne peut PAS “coller” tout seul du texte sur la photo depuis une liste.**  
(La fonction “Insert Text” est manuelle.)

👉 **MAIS digiKam peut injecter automatiquement les noms comme *étiquettes de visages* (face tags)**, ce qui est souvent ce que les gens appellent “injecter”.  
Ensuite vous pouvez exporter une image **avec ces noms superposés**, sans devoir les taper un par un.

---

# ✔️ Solution 1 — Injecter les noms en tant que *Face Tags* (automatique)
C’est la méthode la plus propre.

## 🔧 Étapes
1. **Ouvrez digiKam**
2. Allez dans la barre latérale gauche → **People**
3. Cliquez sur **Detect Faces**  
   digiKam va détecter les visages.
4. Pour chaque visage détecté, cliquez sur le tag vide → tapez le nom  
   (Daniele, Alessandro, etc.)
5. digiKam enregistre les noms :
   - dans sa base interne  
   - dans les métadonnées XMP/IPTC (si activé)

📌 *Vous n’avez plus besoin d’écrire les noms sur la photo : ils sont “injectés” dans l’image en tant que données.*

---

# ✔️ Solution 2 — Exporter l’image AVEC les noms inscrits automatiquement
Une fois que les visages sont tagués :

1. Sélectionnez la photo  
2. Cliquez **Export → Export with overlays**  
   (ou dans certains thèmes : **Tools → Export → Image Overlays**)
3. Cochez **Show Face Tags**
4. Choisissez :
   - Taille du texte  
   - Position (au-dessus ou sous le visage)  
   - Couleur du texte  
   - Ajout d’un cadre ou non  
5. Exportez : digiKam génère une **nouvelle photo avec les noms inscrits automatiquement**, grâce aux face tags.

👉 **Avantage énorme :**  
Vous n’éditez pas l’image originale et vous pouvez changer le style à tout moment.

---

# ✔️ Solution 3 — Injection par script externe (rare mais possible)
Si vous avez une **liste des noms + coordonnées des têtes**, on peut injecter automatiquement dans digiKam via :

- un fichier **XMP sidecar**  
- un script [[Python]] utilisant **exiv2**  

Mais uniquement si vous souhaitez automatiser complètement (gros lot de photos).

Si cela vous intéresse, je peux vous générer :
- un **script [[Python]] prêt à l’emploi**,  
ou  
- un **fichier XMP modèle** dans lequel on injecte les noms des 4 rangées.

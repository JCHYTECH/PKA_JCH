## Résumé

Pep Ventosa est un photographe au style graphique proche du cubisme : il superpose plusieurs points de vue d'un même sujet en une seule image, créant un effet de fondu onirique.

La technique est étonnamment simple :
- **À la prise de vue** : tourner autour d'un sujet en arc de cercle, déclencher 10 à 100+ fois
- **Au post-traitement** : empiler les images en calques, réduire l'opacité à ~14%, jouer avec les modes de fusion, nettoyer les artéfacts aux masques

Le workflow est quasi identique sur Photoshop et Affinity Photo. Résultat accessible même avec un smartphone — le sujet et le mouvement font tout.

---

## Procédure

### Phase 1 — Prise de vue

| Étape | Action |
|-------|--------|
| 1 | Choisir un **sujet statique** (arbre, bâtiment, statue, rocher…) |
| 2 | Se positionner face au sujet, **cadrer large** |
| 3 | Décrire un **arc de cercle** en avançant autour du sujet |
| 4 | Déclencher **10 à 100+ photos** en continu — Pep Ventosa en utilise jusqu'à 100, 10–18 suffisent pour un bon résultat |
| 5 | Garder le sujet **centré** dans le cadre tout au long du mouvement |

**Matériel :** Smartphone ou appareil photo — aucun setup complexe nécessaire.

### Phase 2 — Empilement (Photoshop / Affinity)

| Étape | Action |
|-------|--------|
| 1 | **Ouvrir toutes les photos comme calques** dans un même document |
| 2 | **Calque du bas = référence** — opacité 100%, ne pas y toucher |
| 3 | Sélectionner **tous les autres calques** (Shift+clic du haut jusqu'au 2ᵉ) |
| 4 | Régler l'**opacité à ~14%** (ajuster selon l'effet recherché) |
| 5 | Tester les **modes de fusion** : Produit, Superposition, Incrustation… |

### Phase 3 — Nettoyage et finitions

| Étape | Action |
|-------|--------|
| 1 | Identifier les **artéfacts gênants** (panneaux, passants, branches parasites…) |
| 2 | Sur le calque concerné, créer un **masque de fusion** |
| 3 | **Pinceau noir**, dureté **0%**, taille assez large |
| 4 | ⚠️ Bien vérifier que le **masque est sélectionné** (cadre blanc autour), pas le calque |
| 5 | Peindre sur l'élément indésirable → il disparaît |
| 6 | Répéter pour chaque calque où l'artéfact apparaît |
| 7 | **Recadrer** si les bords présentent des décalages (crénelage) |
| 8 | Sauvegarder |

### Variations créatives

- Changer le **mode de fusion** par calque (pas tous identiques)
- Faire un **deuxième passage** avec un mode de fusion différent par-dessus
- Jouer sur des **opacités variables** entre les calques
- Ajouter un calque de **réglages global** (courbes, saturation) en fin de stack

# 1) Référentiel éthologique Anatidae (V0.9)

## 1.1 Principes (académique, exploitable)
- **Deux types de comportements** (standard en éthologie appliquée) :
  - **STATE** (état durable) : ex. *resting*, *foraging*, *preening* — utile pour budgets d’activité et contexte.
  - **EVENT** (événement bref) : ex. *peck*, *chase*, *mount* — utile pour narration et séquences clés.  
  Cette logique est utilisée dans des travaux récents de budgets d’activité et d’éthogrammes standardisés chez les anatidés en contexte zoologique. citeturn0search4

- **Trois couches par entrée** :
  1) **Code stable (EN)**  
  2) **Intitulé scientifique EN/FR**  
  3) **Traduction éditoriale (FR)**

- **Structure du code**
  ```
  ANAT-[DOMAIN]-[SUBDOMAIN]-[DETAIL]-[MOD]
  ```
  - `ANAT` = périmètre (Anatidae)
  - `DOMAIN` = grand domaine (SOC/AGG/CRT/COP/PAR/FOR/VOC/MAI/MOV/ANT/HUM/INT)
  - `MOD` = modificateurs optionnels (SEX=M/F/U ; AGE=AD/JUV ; SPEC=<species> ; CONTEXT=breeding/winter ; INTENS=low/med/high)

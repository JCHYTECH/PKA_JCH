# WildNexus — Data

Ce répertoire accueillera les données projet liées à P0 et aux phases suivantes.

## Contenus attendus

- exports de mesures RF ;
- logs EVT terrain ;
- données de consommation énergétique ;
- résultats de benchmark caméra ;
- jeux de test du classifieur animal / non-animal ;
- exports structurés compatibles Camtrap-DP ou schéma WildNexus ;
- registres composants et fournisseurs si exportés depuis tableur.

## Règles

- Ne pas déposer de données personnelles identifiables sans base légale et traitement RGPD explicite.
- Ne pas déposer d'images de personnes non floutées.
- Documenter la source, la date, le protocole de collecte et le statut d'usage.
- Préférer des sous-répertoires datés ou nommés par protocole.

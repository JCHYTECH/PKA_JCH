Sources integrees :

Objectif du document : constituer une base de depart pour definir la mission Artheon/Nuances, structurer l'offre B2B, preparer le dispositif commercial et technique, et identifier les points a clarifier avant formalisation du plan projet.

## 1. Synthese de cadrage

Nuances se positionne comme le "copilote energie" des PME, industries et clients professionnels. La societe ne doit pas etre presentee uniquement comme un installateur photovoltaïque ou batterie, mais comme un operateur d'optimisation energetique capable d'agir sur plusieurs leviers :

- audit et diagnostic energetique ;
- collecte de donnees via boitier, sensors ou gateway ;
- pilotage intelligent via EMS / Nuances Brain ;
- installation PV, batteries industrielles, bornes et actifs energetiques ;
- optimisation de l'autoconsommation ;
- peak shaving, demand shifting et time-of-use ;
- trading electricite, reserves et flexibilite ;
- valorisation de l'injection et contrats energie ;
- leasing, tiers investissement et financement de projets ;
- reduction des couts energetiques et des emissions CO2.

La reunion a confirme que la mission attendue ne doit pas etre theorique. Elle doit produire une strategie technico-commerciale utilisable par Nuances pour lancer ou accelerer son activite B2B, preparer le travail de la force commerciale, organiser le travail technique en aval, suivre les KPI en temps reel et permettre l'ajustement des strategies commerciales et operationnelles.

Point central : la mission doit permettre de reduire les temps de prospection et d'augmenter la qualite des rendez-vous commerciaux. Le vendeur doit pouvoir arriver rapidement avec une hypothese de solution, une estimation de rentabilite et une liste precise de donnees a valider.

## 3. Proposition de valeur Nuances

### 3.1 Promesse globale

La promesse principale est la suivante :

> reduire les couts energetiques, reduire les emissions CO2 et creer de nouveaux revenus ou gains d'optimisation par le pilotage intelligent des actifs energetiques.

Dans la presentation, cette promesse est exprimee autour de reductions pouvant aller jusqu'a 80 % dans certains cas combines. Ce chiffre doit etre securise : il doit etre rattache a des cas d'usage precis, des hypotheses claires et des limites de validite.

### 3.2 Positionnement recommande

Positionnement a privilegier pour le B2B :

> Nuances aide les PME et industries a transformer leur site en actif energetique pilote, mesurable et rentable.

Ce positionnement est plus fort qu'une approche "vente de panneaux" ou "vente de batteries". Il permet de relier :

- la donnee ;
- les equipements ;
- le logiciel ;
- le financement ;
- le trading/flexibilite ;
- la performance financiere ;
- la decarbonation.

### 3.3 Preuves commerciales existantes

La presentation met en avant :

- plus de 4 000 installations realisees ;
- plus de 10 MEUR de ventes en 2025 ;
- 4 MEUR au dernier trimestre ;
- experience des fondateurs dans le solaire, l'energie, le trading et l'installation ;
- capacite d'investissement annoncee via Nuances Energy ;
- expertise PV/BESS/EMS/sensors.

Ces preuves doivent etre structurees dans un "pack credibilite" pour les rendez-vous B2B.

## 4. Architecture des offres B2B

### 4.1 Audit energie

Role :

- analyse des actifs, consommations, productions et couts energie ;
- base de calcul pour economies kWh, CO2, ROI, cash-flow et devis ;
- support du dossier preliminaire vendeur.

Valeur B2B :

- transforme un prospect en cas d'usage chiffre ;
- reduit le temps de qualification ;
- permet de prioriser les prospects selon potentiel economique ;
- sert de pont entre commercial, technique et financement.

Points a definir :

- donnees minimales necessaires ;
- contenu standard de l'audit ;
- format du rapport ;
- delai de production ;
- niveau de precision attendu avant et apres rendez-vous.

### 4.2 Boitier / sensors / gateway de collecte

La reunion insiste sur l'existence d'un boitier permettant de capturer les donnees utiles au pilotage energetique. La presentation parle aussi de Sensors et de monitoring.

Role strategique :

- offre d'entree B2B ;
- outil de diagnostic ;
- base de preuve rapide ;
- outil de collecte de donnees ;
- declencheur d'upsell vers Brain, BESS, PV, contrats energie ou leasing ;
- support possible a l'IA pour detection de patterns, anomalies et optimisations.

Promesse :

- rapports personnalises ;
- alertes et detection de dysfonctionnements ;
- identification des pics et inefficacites ;
- reduction potentielle des consommations de 10 a 15 % ;
- objectivation des gains.

Questions a clarifier :

- boitier proprietaire ou solution partenaire ?
- donnees capturees exactement ;
- temps d'installation ;
- cout du boitier ;
- abonnement ou vente unique ;
- qui analyse les donnees ;
- integration CRM/application technique.

### 4.3 Nuances Brain / EMS

Role :

- piloter les assets energetiques ;
- reduire les pics de demande ;
- decaler les consommations ;
- acheter ou consommer au bon moment ;
- proposer des scenarios d'optimisation ;
- devenir la couche logicielle recurrente de la relation client.

Cas d'usage :

- peak shaving ;
- demand shifting ;
- time-of-use ;
- pilotage BESS ;

- compatibilite equipements ;
- supervision humaine ou automatisation ;

- modele economique : abonnement, forfait, partage de gains, success fee.

### 4.4 BESS - batteries industrielles

Promesse :

- stocker l'electricite ;
- optimiser l'autoconsommation ;
- generer des revenus via trading, reserves et arbitrage ;
- reduire les couts via peak shaving ;
- valoriser les sites avec cabine electrique et point de connexion pertinent.

Cas d'usage :

- autoconsommation ;
- trading wholesale ;
- reserves / equilibrage reseau ;
- peak shaving ;
- flexibilite reseau.

Exemples de revenus issus de la presentation :

- 100 kW / 200 kWh : 10 000 a 25 000 EUR/an ;
- 200 kW / 400 kWh : 20 000 a 50 000 EUR/an ;
- 300 kW / 600 kWh : 30 000 a 75 000 EUR/an ;
- 500 kW / 1 000 kWh : 50 000 a 125 000 EUR/an.

Pre-requis identifies :

- cabine moyenne ou haute tension ;
- puissance disponible ;
- point de connexion ;
- capacite reseau ;
- espace disponible ;
- cadre securite/incendie ;
- EMS/BMS ;
- modele de financement.

Importance pour le B2B :

- offre probablement prioritaire dans la strategie d'investissement ;
- forte valeur differenciante ;
- necessite un screening rigoureux pour eviter les prospects non rentables ;
- doit etre reliee a un modele financier defensible.

### 4.5 Nuances Sun - PV, batteries et bornes

Role :

- reduire les emissions CO2 ;

### 4.6 Nuances Energy - contrats, injection et pool energie

Promesse :

- creer un pool d'energie verte entre producteurs et consommateurs ;
- racheter l'injection a de meilleures conditions ;
- vendre une energie verte a prix stable ;
- reduire la dependance au marche ;
- matcher surplus et consommation entre entreprises.

Valeur B2B :

- outil de fidelisation ;
- source potentielle de revenus recurrents ;

Points a securiser :

## 5. Cibles B2B prioritaires

### 5.1 Cibles naturelles

Les cibles B2B les plus pertinentes a ce stade :

- PME industrielles avec consommation electrique significative ;
- PME industrielles familiales ou a chaine courte de decision ;
- sites avec cabine moyenne ou haute tension ;
- sites avec forte puissance appelee ;
- sites avec pics couteux ;
- entreprises avec toiture ou terrain disponible ;
- entreprises deja equipees en PV avec injection excedentaire ;
- sites logistiques, froid, agroalimentaire, production, entrepots, retail multi-sites ;
- entreprises avec objectifs CO2 ou contraintes de decarbonation ;
- producteurs ou consommateurs pouvant entrer dans un pool energie.

### 5.2 Cibles a eviter ou de-prioriser

A definir avec Nuances, mais probablement :

- sites sans donnees disponibles et faible facture energie ;
- prospects sans decisionnaire accessible ;
- entreprises avec cycle d'achat trop long pour la phase de lancement ;
- sites sans cabine ni potentiel PV/BESS/EMS ;
- clients cherchant uniquement le prix le plus bas sans logique de pilotage ;
- projets soumis a blocages techniques ou juridiques lourds.

### 5.3 Cible de depart recommandee

Pour une premiere phase B2B efficace :

> PME industrielles familiales, avec cabine MT/HT, facture energie significative, decision rapide, consommation pilotable ou potentiel PV/BESS.

Cette cible est coherente avec la reunion : elle combine potentiel technique, valeur economique et facilite relative de decision.

## 6. Screening et qualification commerciale

### 6.1 Principe

Le screening doit permettre de savoir rapidement si un prospect merite :

- un simple rejet ;
- un appel de qualification ;
- un rendez-vous commercial ;
- une installation de boitier/sensors ;
- un audit approfondi ;
- une offre BESS/PV/Brain/Energy.

### 6.2 Donnees minimales a collecter

Donnees recommandees :

- nom de l'entreprise ;
- secteur ;
- adresse du site ;
- type de decisionnaire ;
- chaine de decision courte ou longue ;
- contrat de fourniture electricite ;
- consommation annuelle kWh ;
- cout annuel energie ;
- puissance de cabine ;
- puissance maximale appelee ;
- courbe quart-horaire si disponible ;
- PV existant ;
- injection excedentaire ;
- surface toiture/terrain ;
- horaires d'activite ;
- equipements pilotables ;
- responsable energie present ou non ;
- objectif client : economie, CO2, independance, revenus, stabilite prix ;
- appetence financement : achat, leasing, tiers investisseur.

### 6.3 Scoring initial propose

Critere 1 - Potentiel technique :

- cabine MT/HT ;
- puissance disponible ;
- consommation significative ;
- PV existant ou potentiel ;
- actifs pilotables.

Critere 2 - Potentiel economique :

- facture energie elevee ;
- pics de puissance ;
- injection mal valorisee ;
- potentiel economie visible ;
- capacite de financement ou leasing.

Critere 3 - Accessibilite commerciale :

- decisionnaire identifiable ;
- chaine courte ;
- urgence ou douleur claire ;
- ouverture a l'audit ou au boitier ;
- secteur prioritaire.

Critere 4 - Faisabilite operationnelle :

- donnees disponibles ;
- contraintes techniques moderees ;
- site accessible ;
- delai compatible ;
- risque juridique/contractuel acceptable.

Sortie du scoring :

- A : priorite immediate, rendez-vous avec dossier preliminaire ;
- B : qualification complementaire ou boitier/sensors ;
- C : nurturing ou attente de donnees ;
- D : hors cible.

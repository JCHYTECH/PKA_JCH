# PILOTAGE_STRATEGIQUE — Index dossier

Dernière mise à jour : 2026-05-13

## Rôle

Dossier de pilotage transverse du portefeuille JCH.

Ici se décident :
- les priorités entre projets
- les arbitrages de focus
- les objectifs à 90 jours
- les KPI de portefeuille
- les risques, frictions et opportunités transverses

Ce dossier ne remplace pas `03_PROJECTS/`.
Il sert de couche de supervision entre :
- `03_PROJECTS/` — exécution par projet
- `05_CONTEXT_JCH/` — contexte personnel et identité JCH

## Structure

| Fichier | Rôle |
|---|---|
| `priorites-actives.md` | Vue courte des priorités réellement en cours |
| `roadmap-90-jours.md` | Horizon tactique portefeuille |
| `arbitrages-decisions.md` | Décisions de focus, stop, report, renforcement |
| `kpi-portefeuille.md` | Indicateurs transverses de pilotage |
| `opportunites-a-suivre.md` | Pistes à surveiller sans polluer l'exécution |
| `risques-et-frottements.md` | Points de tension, dette, obstacles systémiques |

## Principes

- Un sujet ne monte ici que s'il concerne plusieurs projets ou le portefeuille global.
- Les détails opérationnels restent dans les dossiers projet.
- Les décisions prises ici doivent pointer vers les projets concernés.

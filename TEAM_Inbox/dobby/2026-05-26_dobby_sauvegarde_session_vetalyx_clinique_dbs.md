# Sauvegarde session - VETALYX clinique, tests rapides et DBS

## Demandes traitees

1. Analyse clinique des allergies chien/chat pour test rapide.
2. Classement des allergies IgE alimentaires, respiratoires et cutanees.
3. Classement de prevalence/priorite allergenes Europe.
4. Specification de tests rapides 6 lignes pour chien et chat.
5. Rapport final distribuable partenaires science/technique.
6. Protocole de validation clinique a proposer a deux KOL :
   - Dr Morgane Cannone, Maisons-Alfort ;
   - Dr Billen, ULg.
7. Analyse de l'utilisation du buvard / DBS pour conservation et tests IgE.
8. Individualisation du DBS comme sous-protocole analytique a valider avec les partenaires chinois.
9. Creation de logigrammes Mermaid.
10. Creation d'une liste de taches operationnelle.

## Decisions principales

- Le test rapide VETALYX doit etre positionne comme outil de triage/orientation clinique, pas diagnostic autonome.
- La priorite produit est l'IgE environnementale/dermique chien-chat.
- Les panels alimentaires doivent rester prudents : support de sensibilisation et aide a l'eviction, pas diagnostic d'allergie alimentaire.
- Les lignes allergenes doivent etre des mixes lorsque le format est limite a 6 lignes.
- Le serum/plasma reste la matrice de reference pour comparer les tests IgE.
- Le DBS/buvard est prometteur, mais uniquement comme sous-etude de pontage analytique.
- Le DBS sera conserve a basse temperature :
  - sechage 2-4 h a temperature ambiante controlee ;
  - 2-8 C apres ensachage ;
  - -20 C si conservation >72 h ;
  - -80 C pour biobanque si disponible.
- Le succes ou l'echec du DBS ne doit pas bloquer l'evaluation clinique du test rapide.

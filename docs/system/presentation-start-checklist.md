# Checklist — Demarrage presentation

Utiliser cette checklist avant de produire un PowerPoint, un deck, une note visuelle ou tout autre support de presentation.

## 1. Cadre visuel

- identifier la marque ou le projet de reference ;
- verifier s'il existe deja une charte graphique ;
- verifier s'il existe deja un layout, un template, un mockup ou un deck precedent reutilisable ;
- ne pas repartir de zero si un systeme visuel existe deja.

## 2. Sources a ouvrir d'abord

- charte graphique du projet ou de la marque ;
- README ou INDEX du projet ;
- layouts et variantes de slides existants ;
- livrable de reference si un document maitre existe deja.

## 3. Decision de structure

- definir le type de support : analytique, narratif, commercial, dirigeant, statutaire ;
- choisir le systeme de layout correspondant ;
- fixer le nombre de slides ou le niveau de densite avant de rediger.

## 4. Regles de production

- une idee maitresse par slide ;
- la hierarchie visuelle prime sur la quantite de texte ;
- les logos et signatures de marque doivent etre presents avec discipline, pas de facon invasive ;
- les chiffres, sources et preuves doivent etre visibles aux endroits critiques.

## 5. Verification finale

- verifier que le support respecte la charte ;
- verifier que le support reutilise bien les layouts prevus ;
- verifier qu'aucune slide ne ressemble a un gabarit generique hors marque ;
- verifier qu'il existe une version de reference clairement identifiable.
